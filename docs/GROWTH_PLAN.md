## 🎯 Phase 3: User Acquisition (May-June)

### Organic Growth Channels

**1. App Store Optimization (ASO)**
- Target keywords: "events near me", "things to do", "concerts tonight", "local events"
- A/B test icons, screenshots, descriptions
- Encourage reviews (in-app prompt after 5th swipe)
- **Goal: 10,000 organic installs/month**

**2. SEO (Web)**
- Event pages indexed by Google (200K+ pages)
- City landing pages ("Events in Miami", "Things to do in Austin")
- Blog content ("Best festivals 2026", "Summer concerts guide")
- **Goal: 50,000 organic web visitors/month**

**3. Social Media**
- Instagram: Event photos, reels, stories
- TikTok: "Hidden gem events in [city]" format
- Twitter: Event announcements, trending topics
- **Goal: 25,000 followers across platforms**

**4. Viral Mechanics (Built-in)**
- [ ] "Share with friends" after swiping right
- [ ] "X friends are going!" notifications
- [ ] Referral program ($5 credit per invite)
- [ ] Group planning feature ("Create a squad")
- **Goal: 1.5 viral coefficient (each user brings 1.5 more)**

### Paid Acquisition (Later, after PMF)

**Target CAC:** $1-3 per install
- Facebook/Instagram ads (event interest targeting)
- Google UAC (app install campaigns)
- TikTok ads (younger demo)
- Influencer partnerships (micro: 10K-100K followers)

**Budget:** Start $5K/month → scale to $50K/month if ROAS >3x

# Eventgasm Business Plan
*Last Updated: March 2026*

## Executive Summary

**Eventgasm** is a nationwide event discovery app that aggregates events from 14+ sources into one searchable platform. Unlike competitors (Ticketmaster, Eventbrite, Facebook Events), we show ALL events regardless of where tickets are sold.

**Value Proposition:** "Never miss what's happening near you."

---

## Revenue Model

### Primary Revenue Streams

| Stream | % of Revenue | How It Works |
|--------|--------------|--------------|
| **Affiliate Commissions** | 50% | 5-15% on ticket sales via our links |
| **Advertising** | 35% | AdMob/Meta ads in-app |
| **Premium Subscription** | 15% | Ad-free + extra features |

### Affiliate Partners (Apply in Year 1)
- Ticketmaster Affiliate Program (5-8%)
- SeatGeek Affiliates (6-10%)
- Eventbrite Affiliates (5%)
- StubHub Affiliates (8-12%)
- VividSeats (10-15%)

### Premium Subscription ($2.99/month)
- Ad-free experience
- Early event alerts
- Exclusive presale codes
- Export to calendar (bulk)
- Price drop notifications

## Marketing Plan

### Phase 1: Launch (Months 1-3)
**Goal:** 500 users, validate product-market fit
**Budget:** $0 (organic only)

| Tactic | Action | Expected Users |
|--------|--------|----------------|
| **Friends & Family** | Personal outreach | 50 |
| **Reddit** | Post in r/Orlando, r/Miami, local subs | 100 |
| **Facebook Groups** | "Things to do in [City]" groups | 150 |
| **App Store SEO** | Keywords: "events near me", "local events" | 100 |
| **Product Hunt** | Launch post | 100 |

### Phase 2: Traction (Months 4-6)
**Goal:** 2,000 users
**Budget:** $200/month

| Tactic | Action | Expected Users |
|--------|--------|----------------|
| **TikTok/Reels** | "Hidden events in [City]" content | 500 |
| **Instagram** | Event highlight reels | 300 |
| **Local Influencers** | Partner with city guides | 400 |
| **App Store Ads** | $100/month test | 200 |
| **Cross-promotion** | Partner with event organizers | 300 |

### Phase 3: Growth (Months 7-12)
**Goal:** 10,000 users
**Budget:** $500/month

| Tactic | Action | Expected Users |
|--------|--------|----------------|

| **SEO Website** | Blog: "Best events in [City] this week" | 1,500 |
| **Email Marketing** | Weekly event digests | 1,000 |
| **Referral Program** | "Invite friends, get premium free" | 1,500 |
| **PR/Press** | Local news, tech blogs | 1,000 |

### Phase 4: Scale (Year 2+)
**Goal:** 50,000+ users
**Budget:** $2,000/month

| Tactic | Action |
